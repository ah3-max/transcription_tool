- [ ] **ASR 串流客戶端 `services/asr.py`（新檔）**

  - 雷（PoC 實證）：握手序列 `session.update → commit(final=False) 啟動 → append → commit(final=True)`，**漏 `commit(final=False)` 會收不到任何 delta**；每個 delta 帶 `language {lang}<asr_text>` 前綴，**串流路要自己剝**（批次端點 server 端才會剝）。
  - 完成判準：丟一段中文音框能拿到去前綴的增量逐字稿。
