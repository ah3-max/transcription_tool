## 1. 資訊架構（Information Architecture）

### 1.1 頁面結構
```
語音轉文字工具（單一應用，左側導覽）
├── 即時翻譯 ── 設定畫面 → 專屬即時舞台（含原文側欄）
├── 錄音記錄 ── 場次列表 → 單場檢視（逐字稿／摘要分頁）
├── 批次逐字稿 ── 新增轉入（上傳→選項→進度→結果預覽）／歷史記錄
├── 文件生成 ── 選來源 → 選範本 → 輸出預覽
└── 一般設定 ── 模型路由 / 資源管理 / 介面語言 / 保留政策
```

### 1.2 導覽邏輯
- 主導覽：**左側固定 sidebar**（桌機），手機改**底部 tab bar**。
- sidebar 固定不隨內容捲動，僅右側主內容區捲動。
- 返回邏輯：各功能為獨立分頁，切換即重置捲動位置；單場／結果為頁內狀態切換，不另開路由。
- 深層連結：Phase 1 不需 URL 深連（單機單頁）。

### 1.3 導覽決策紀錄
| 決策 | 選擇 | 原因 | 備選方案 |
|---|---|---|---|
| 主導覽 | 左側 sidebar | 功能模組固定 5 個、需常駐 | 頂部 nav（高齡使用點擊區較小） |
| 即時翻譯呈現 | 設定→專屬舞台兩段式 | 投影／展示需要乾淨大字畫面 | 單頁混合（資訊太雜） |
| 檔案管理 | 不獨立成頁 | 產出就近放在各功能頁、靠 DB 索引找 | 獨立檔案總管頁（重複且易亂） |

## 3. 互動狀態

### 3.1 全域模式
- **載入：** 摘要與批次結果產生時顯示 spinner＋文字（如「統整中…」「轉錄中…」）；按鈕等待時禁用避免重複點。
- **成功：** 匯出／儲存成功以頂部 toast（綠、3 秒消失）。
- **錯誤：** 上傳格式／大小不符 → 拖放區下紅字；API 失敗 → 頂部 toast（紅、需手動關）。
- **空狀態：** 列表無資料顯示說明＋主操作（如「尚無錄音，先到即時翻譯開始一場」）。
- **確認：** 刪除前 Modal 確認「此操作無法復原」。
- **降級：** 資源不足以即時翻譯 → 橫幅提示「目前資源不足，已切換為手動錄音」。

### 3.2 元件狀態
**Button**：Default 主色白字／Hover 加深／Disabled 灰底 not-allowed／Loading spinner。
**Input/Select**：Default 灰框／Focus 主色框淡底／Error 紅框＋下方紅字／Disabled 淺灰。
**即時舞台譯文**：interim 半透明（草稿）／final 實心（定稿）。

## 5. API 介面設計

### 5.1 總覽
| ID | Method | Path | 用途 |
|---|---|---|---|
| API-01 | POST | /api/jobs | 建立批次工作（上傳音檔） |
| API-02 | GET | /api/jobs | 批次工作清單 |
| API-03 | GET | /api/jobs/{id} | 單一工作（含進度、產出索引） |
| API-04 | GET | /api/jobs/{id}/export?fmt=docx\|md\|txt&lang=zh\|th\|en | 匯出 |
| API-05 | DELETE | /api/jobs/{id} | 移出佇列／刪除 |
| API-06 | WS | /ws/live | 即時串流（音框上行；partial/final＋多語譯文下行） |
| API-07 | GET | /api/sessions、/api/sessions/{id} | 錄音場次清單／單場 |
| API-08 | POST | /api/sessions/{id}/retranslate | 重新翻譯 |
| API-09 | POST | /api/sessions/{id}/summary | 產生摘要 |
| API-10 | POST | /api/records | 由來源＋範本生成文件 |
| API-11 | GET/POST/DELETE | /api/endpoints | 模型路由端點 CRUD |
| API-12 | GET | /api/resources | 當前 VRAM/RAM 用量（給頂欄顯示） |

### 5.2 範例細節

**API-01 POST /api/jobs**（multipart）
請求：`files[]`（音檔）、`src_lang`、`out_langs[]`。
成功 202：
```json
{ "job_id":"j_8f2a", "status":"queued", "files":[{"name":"週會_0623.mp3","duration":2895}] }
```
錯誤：400 格式／大小不符 `{"error":"bad_file","message":"僅支援 mp3/mp4/m4a/wav，單檔 ≤ N"}`；503 資源不足 `{"error":"resource","message":"目前資源不足，請稍後再試"}`。

**API-06 WS /ws/live**
下行訊息：
```json
{ "type":"partial|final", "src":"那個 patient 的血壓…", "translations":{ "th":"…", "en":"…" }, "t":"00:12:34" }
```
停止時：`{ "type":"saved", "session_id":"s_19c" }`。

### 5.3 共通慣例
- 路徑前綴 `/api`；即時走 `/ws`。
- 回應外型 `{ data, error?, message? }`（清單含 `pagination`）。
- 應用本體綁 0.0.0.0 對內網開放（本機開發 3600／測試環境 3610，使用者連 主機內網IP:3610）；其餘 3601–3699 留給自有後端；模型端點由 LMS／vLLM／pod 服務（各自埠、建議僅 localhost）；不在 URL query 放敏感資料。

## 6. 資料模型

### 6.1 資料表
**jobs（批次）**
| 欄位 | 類型 | 必填 | 說明 | 索引 |
|---|---|---|---|---|
| job_id | TEXT | ✓ | 主鍵（伺服器產生） | PK |
| original_name | TEXT | ✓ | 原始檔名（僅顯示，不入路徑） | — |
| zone | TEXT | ✓ | uploads | — |
| src_lang | TEXT | ✓ | 辨識語言模式 | — |
| out_langs | TEXT | ✓ | JSON 陣列（zh/en/th） | — |
| status | TEXT | ✓ | queued/running/done/error | INDEX |
| created_at | TS | ✓ | 建立時間 | INDEX |
| expire_at | TS | ✓ | 到期時間（清除掃此欄） | INDEX |
| path | TEXT | ✓ | 落地目錄 | — |

**sessions（即時錄音）**：session_id(PK)、name、langs(JSON)、duration、status、created_at、expire_at(INDEX)、audio_path、transcript_path。
**outputs（產出）**：id(PK)、ref_type(job/session)、ref_id(INDEX)、kind(transcript/translation/summary/record)、lang、fmt、path、created_at。
**endpoints（模型路由）**：id(PK)、name、url、model、function(asr/batch_tr/live_tr/post)、active。

### 6.2 關聯
```
jobs 1───N outputs
sessions 1───N outputs
endpoints（獨立，被路由設定引用）
```

## 8. 設計決策記錄（Decision Log）
| # | 決策 | 選擇 | 原因 | 備選 |
|---|---|---|---|---|
| D-01 | 主題 | 愛愛院版 v6 為唯一定稿 | 機構品牌一致、高齡友善 | 現行 teal 版（停用） |
